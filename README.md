# Industrial AI Assistant

A local Industrial AI Assistant designed for manufacturing environments.

## Features

- PDF manual question answering
- Alarm troubleshooting
- Semantic search
- Retrieval-Augmented Generation
- FastAPI backend
- Ollama local LLM
- ChromaDB vector database
- Docker deployment

## Target Use Case

This project simulates an AI Assistant integrated with an industrial machine controller, helping engineers and operators search manuals, understand alarms, and retrieve troubleshooting instructions.

## Tech Stack

- Python
- FastAPI
- Ollama
- ChromaDB
- Local embedding model
- Docker
